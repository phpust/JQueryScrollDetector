# JQueryScrollDetector
This is a plugin for jquery to add mouse scroll up and down event 

### Usage
```javascript
	$(document).on('scrollUp',function(){
        alert("scroll up event");
    });
	$(document).on('scrollDown',function(){
        alert("scroll down event");
    });
```
